What is Nenufaar ?

Nenufaar is a simple shell script that performs DNA NGS analyses. It is able to run:

  • On amplicon or capture data
  • On gene panels, exomes (WES), genomes (WGS)
  • A nearly fully paralellised workflow, even for GATK tools (using Queue)
  • On a HPC environment through SLURM or on a single UNIX machine (Linux, MacOS).

Nenufaarity is a singularity container: a single file that contains (nearly) all you need to run nenufaar.
